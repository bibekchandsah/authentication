Features
--------------------------------------------------------------------------------------
Microsoft Authenticator integration with TOTP (Time-based One-Time Password)
The .secret-key.json file contains your TOTP secret
Just start the server and login with your 6-digit codes
No need to reconfigure your authenticator app!
Session-based authentication
Click "üîÑ Regenerate Secret Key" or Delete .secret-key.json file to generate a new secret key
6 Separate Input Boxes: Each digit has its own input field
Numbers Only: Automatically filters out non-numeric characters
Double-Click Clear: Double-click any box to clear all inputs
Auto-Submit: No need to click login - submits automatically
Max Attempts: 5 failed attempts before lockout
Initial Lockout: 15 minutes
Progressive Penalty: Doubles with each violation (max 24 hours)
Rate Limit Dashboard: View all monitored IPs and their status
Clear All Limits: Emergency function to clear all rate limits
